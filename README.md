## Getting Started

1. Clone the repository:

```sh
git clone https://github.com/sufiyancode/portfolio-website.git
```

2. Install dependencies

```sh
cd portfolio-website
npm install
```

3. Start development server:

```sh
npm run dev
```

4. Build for production:

```sh
npm run build
```
